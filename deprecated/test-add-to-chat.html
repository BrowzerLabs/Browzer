üî¨ Technical Information

The add to chat functionality works by:

  1. Injecting JavaScript into each webview when it loads
  2. Listening for mouseup events to detect text selection
  3. Showing a positioned button near the selected text
  4. Sending the selected text via postMessage when clicked
  5. Processing the message in the main renderer to add text to chat input

This test page helps verify that the selection detection, button positioning, and message passing all work correctly.

üêõ Troubleshooting

If the add to chat functionality isn't working:

  1. Check the browser console for error messages
  2. Make sure the page has fully loaded
  3. Try selecting different amounts of text (minimum 5 characters)
  4. Test the keyboard shortcut Ctrl+Shift+A as an alternative
  5. Verify that the chat interface is properly initialized
