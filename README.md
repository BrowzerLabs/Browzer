## 🚀 **Get Started in 60 Seconds**

### Option 1: 💎 **Download & Run** (Recommended)
```bash
# macOS
curl -L https://releases.browzer.com/latest/browzer-macos.dmg -o browzer.dmg
open browzer.dmg

# Windows  
curl -L https://releases.browzer.com/latest/browzer-windows.exe -o browzer.exe
./browzer.exe

# Linux
curl -L https://releases.browzer.com/latest/browzer-linux.AppImage -o browzer.AppImage
chmod +x browzer.AppImage && ./browzer.AppImage
```

### Option 2: 🛠️ **Build from Source** (For Developers)
```bash
git clone https://github.com/BrowzerLabs/browzer.git
cd browzer
npm install && npm run dev
# 🎉 Browzer opens automatically!
```

### Option 3: 🐳 **Docker** (One Command)
```bash
docker run -p 3000:3000 browzer/browzer:latest
# Visit http://localhost:3000
```
